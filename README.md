# MathIsNotMathing
## By Fernando R. and Yonatan D.
### CIS 103-S2023-FinalProject

### What is it?
This is a calculator for creating a function for rational equations with a discontinuance from an already existing graph. This calculator also visualizes the eqation to make sure it matches with what you started with.

### How to use it?
 As it runs it will ask a question weather your graph has only a vertical asymptote or a vertical and horizontal asymptote. It will then activate one of the functions and ask for your input. Once you have inputed the values it will then create your rational equation while also creatig the corresponding graph.

 ### Requirements to run:
 - Python 3
