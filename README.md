### What is it?
MathIsNotMathing is a Python-based calculator designed to generate and visualize rational equations based on user-defined asymptotes and discontinuities. The program allows users to input whether their graph includes only a vertical asymptote or both vertical and horizontal asymptotes, and then constructs a corresponding rational function. Using user inputs such as intercepts and discontinuities, the program dynamically builds the equation and graphs it for visualization.

The project was developed using Python, leveraging libraries such as NumPy for handling numerical data and Matplotlib for graphing the equations. One of the main challenges was correctly modeling discontinuities and ensuring the graph accurately reflects holes and asymptotic behavior. Additionally, handling user input and converting it into valid mathematical expressions required careful validation and debugging.

This tool helps students better understand how rational equations behave and how asymptotes and discontinuities affect graphs.

### Features
Generates rational equations based on user input
Supports:
Vertical asymptotes
Horizontal asymptotes
Discontinuities (holes)
Graphs equations in real time
Saves graph output as an image

### How to use it?
 As it runs it will ask a question weather your graph has only a vertical asymptote or a vertical and horizontal asymptote. It will then activate one of the functions and ask for your input. Once you have inputed the values it will then create your rational equation while also creatig the corresponding graph.

Run the program in Python
Choose the type of graph:
Option 1: Vertical asymptote with discontinuity
Option 2: Vertical and horizontal asymptotes
Input the requested values
View the generated graph
